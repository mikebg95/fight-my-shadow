# Sound Assets

## boxing_bell.mp3
A short boxing bell ring sound (1-2 seconds) that plays at the start and end of rounds in Drill and Add-to-Arsenal sessions.

You can find free boxing bell sounds at:
- https://freesound.org/ (search for "boxing bell")
- https://mixkit.co/free-sound-effects/boxing/

Save the sound file as `boxing_bell.mp3` in this directory.
